# vscode-erde

VSCode language extension for the Erde programming language. See CHANGELOG.md
for version changes.

## Features

- [x] Syntax highlighting
- [x] Bracket matching
- [x] Bracket autoclosing
- [x] Bracket autosurrounding
- [x] Comment toggling
- [x] Auto indentation
- [ ] Folding
- [ ] Language server?
